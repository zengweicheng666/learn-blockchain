# 第 3 天：哈希函数（SHA256）

日期：2026-05-12

## 核心概念

### 哈希函数的四个性质
1. **确定性**：同一输入 → 永远返回同一哈希值
2. **快速计算**：输入任意长度的数据，都能在合理时间内计算出哈希
3. **雪崩效应**：输入哪怕改一个字符，输出哈希完全不一样
4. **抗碰撞性**：极难找到两个不同的输入产生相同的哈希值

### 实操记录

#### Node.js 单次测试
```
输入："hello"
哈希值：2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824

输入："hellp"（改一个字符）
哈希值：fdd7585e08c4e2afd71dcabdb4636c89d557a3f42db9e2040c8bbd1708aa4ce7
```

#### 位数差异对比
```
总位数：256 位（SHA256 = 256 bits = 64 hex chars）
不同的位数：131
不同比例：51.2%
```
